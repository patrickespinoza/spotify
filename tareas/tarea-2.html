Encabezado (h1-h6)

Las etiquestas h1- to h6 se utilizan para definir encabezado HTML

h1 define el titulo mas importante

h6 define el encabezado menos importante

Nota: utilice solo uno h1 por pagina este debe representar el titulo/asunto principal de toda la pagina. Ademas, nose salte los niveles de encabezado: comience con h1, luego use h2, y asu sucesivamente
